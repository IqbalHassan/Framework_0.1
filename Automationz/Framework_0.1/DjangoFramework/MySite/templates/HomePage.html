Why We Built It!

Automation Solutionz provides Web Based test case management and execution. Our team is composed of one of the most talented individuals with vast knowledge in software development life cycle.

During our 10+ years of venture in software automation, we have developed framework around QTP, Unit testing, Applescript with Automator, Ration Robot, Robot, Fitness and various other customized framework.

Code based driven testing (unit testing) provides best solution for finding bugs before they are introduced to the product. It allows developers to detect bug at the very early stages of the software development life cycle. It requires decent level coding and good code flow knowledge of the product.

However when it comes to complex testing which requires integration with multiple modules, it does not meet the testing requirements. It also does not produce good reporting matrix and does not keep execution history. It does extremely poor with end user testing as it does not drive the application.

On the other hand, record and playback automation framework (QTP) may provide solution for testers with very little or no coding experience to create automation testing. This type of framework proves to be one of the most expensive when it comes to reliability and maintenance. However, these types of frameworks are focused with end-end testing and provide a better reporting.

Our approach of automation framework combines best of both worlds. We have analyzed all the bad and good of various framework and came up with a solution that would satisfy both developers and testers requirements.

We are able to execute white box testing or black box testing. We achieved this by allowing developers to write unit testing and integrating the unit testing directly into our framework with very little work. Developers are able to run their testing before they submit the code and manual testers also able to write keyword based testing that would drive the UI to simulate end user testing.

Our reporting structure provides one of the most comprehensive reporting in the market. We focused most on our reporting during our research prior to designing the framework. We are able to pin point root case of failure with screen captures and detailed logs that are organized with various log level. This allows testers to view results without going into too much detail of failure reason and at the same time allows developers to drill deep in finding the root cause. All this is done with dynamic table with clear and concise reporting.

We believe that with our intelligent reporting and advanced cloud based test case management, execution and reporting, your software will be able to deliver the highest quality with lowest price point. We are always learning new technology and would enjoy any challenges your developers or testers faces.

Please follow the steps to create a blank db(manually):
1. drop the public schema from postgres db with the option drop cascaded.
2. now create a new schema named public
3. now run the sql file in the following order:
	3.1. run createExtension.sql
	3.2. run createSequence.sql
	3.3. run createTables.sql
	3.4. run createFunction.sql
	3.5. run createView.sql
4. on success you will get a blank db.

5. please register the name and password(manually) in the backend to login.



If you want a blank database,it has been also made for you.The File is:
	FullyBlankDataBase.backup

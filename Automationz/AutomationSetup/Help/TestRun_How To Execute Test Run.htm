How To Execute Test Run?

0. Pre-Req: Test Cases or Set, Machine, Milestones

1. Login as User

2. Under 'Testing' Click on 'Deployment'

3. Enter 'Test Cases or Test Set'

4. Fill 'Details' Tab including Starting/End Dates, Assigned Testers, Emails, Objective, Milestone & Machine

5. Once everything is complete 'Run' button should appear

6. Click on 'Run' Button

7. Once Test Run is executed - Emails should be send out to the testers with details of Test Run
